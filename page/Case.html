Estructura de control Case

Una estructura de control Case (o Switch) es un mecanismo de programación que permite seleccionar y ejecutar uno entre varios bloques de código basándose en el valor de una variable o expresión.

Sirve para simplificar la toma de decisiones cuando hay múltiples condiciones posibles, evitando el uso excesivo de estructuras if-else. Es ideal para manejar opciones claras y predefinidas, como menús, rutas, o estados de una aplicación.
